#### 15-1 Redux基本介绍 (12:24)

几个基本概念

state

action

reducer



actions   // 行为，从图书馆借书。

store  // 存储数据的公共区域，图书馆管理员

reducers   // 图书记录册

react Components // 借书人

 

安装插件

```
yarn add redux --save
yarn add react-redux --save
```

5. 添加 provider 作为项目的根组件，用于数据的存储
